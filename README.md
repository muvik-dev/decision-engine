# decision-engine

### To run the project, follow these 3 steps:

#### 1. Start the frontend by executing the commands below:
 ```$bash
 cd front-end
 yarn install
 yarn serve
 ```

#### 2. Launch the backend by executing the commands below:
 ```$bash
 cd backend
 ```

#### 3. Run Docker by executing the commands below:
 ```$bash
 cd infrastructure
 docker-compose up -d
 ```
